Getting started

This is a starter template for Jekyll 4 with Tailwind CSS 4. Edit _config.yml for site metadata, add pages and collections as needed, and customize styles in css/main.css.

  • make setup — install Ruby gems and npm packages
  • make serve — run the local development server
  • make serve-dev — serve with development overrides
  • make build — build the static site to _site/
